## 문제

**별표 출력하기 20**

삼각형의 높이 n의 값을 입력받아 별표로 이루어진 삼각형을 출력하는 프로그램을 작성해보세요.

다음은 n이 4인 예시입니다.

```
****
***
**
*
```

**Input**
첫 번째 줄에 삼각형의 높이를 나타내는 정수 n이 주어집니다.

**Constraints**
- 1 ≤ n ≤ 10

**Output**
첫 번째 줄부터 n개의 줄에 걸쳐 주어지는 n의 값에 따른 별표들을 예제와 같이 출력합니다.

**I/O Examples**

Example 1
- Input: `5`
- Output:
```
*****
****
***
**
*
```

## 📌 Code Review 📌

### 01. 기존 풀이
```javascript
const n = Number(require("fs").readFileSync("/dev/stdin").toString().trim());

for (let i = n; i > 0; i--) {
    let star = "*".repeat(i);
    console.log(star);
}
```
- 변수명 `star`보다는 `stars`가 의미상 더 맞다.
- `star` 변수를 선언하지 않고 바로 출력해도 된다.

### 02. 개선 방향: 기존 코드 리팩토링
```javascript
const n = Number(require("fs").readFileSync("/dev/stdin").toString().trim());

for (let i = n; i > 0; i--) {
  console.log("*".repeat(i));
}
```

### 03. 다른 풀이: `Array.from` 활용
```javascript
const n = Number(require("fs").readFileSync("/dev/stdin").toString().trim());

Array.from({ length: n }, (_, idx) => n - idx)
  .forEach(i => console.log("*".repeat(i)));
```
- 성능과 가독성 측면에서 `for`문이 더 좋지만, '어떤 배열을 만들지'에 집중하는 데이터 변환 파이프라인으로서 재사용성이 좋다.

### 04. 사전 지식
- `Array.from`: 배열이 아닌 것(유사 배열, 반복 가능한 객체 등)을 진짜 배열로 변환하는 메서드
  ```javascript
  Array.from({ length: 3 }, (_, i) => i * 2)
  // [0, 2, 4]
  ```
